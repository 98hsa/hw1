Döngüde JavaScript

While Döngüsü

whileDöngü uzunluğunda belirli bir koşul doğru olarak kod bloğu döngüsü.

ÖRNEK:

while (condition) {

// code block to be executed

}

Do/ While Döngüsü

do/whileDöngü sırasında döngünün bir çeşididir. Bu döngü, koşulun doğru olup

olmadığını kontrol etmeden önce kod bloğunu bir kez yürütür, ardından koşul doğru

olduğu sürece döngüyü tekrarlar.

ÖRNEK:do {

// code block to be executed

}

while (condition);

var i = 0;
